Most important git commands are:

  1. Git desktop:to navigate to desktop
  2. Git mkdir:Create a new folder
  3. Git touch:Create a new file
  4. Code .+ enter open a code editor

Most important git command to handle github:

  • Git clone URL:clone an empty repository
  • Git add . enter:add all files and folders to stage area
  • Git commit -m "message"
  • Git push:it will push all codes to github
